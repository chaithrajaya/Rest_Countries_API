Project Overview

Countries Explorer is a frontend web application that allows users to browse and explore information about countries around the world.
The application fetches real-time data from the REST Countries API and provides features such as searching, filtering, sorting, dark mode, and a detailed country information modal.

This project is built using HTML, CSS, and JavaScript
